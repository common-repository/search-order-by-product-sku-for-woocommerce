Allows WooCommerce orders to be searched by product SKU - Stock Keeping Unit.

== Description ==

Allows WooCommerce orders to be searched by product SKU.
You can use wildcards (asterisk or underscore) in your search.

= Translations in your language =

* English
* Italian (it_IT)
* Our plugin is fully [WPML Compatible](https://wpml.org/documentation/getting-started-guide/string-translation/).

== Installation ==

Steps to install this plugin.

Be sure you're running WooCommerce 2.0+

1. In the downloaded zip file, there is a folder with name 'WooCommerce-pdf-invoices-italian-add-on'
1. Upload the 'search-order-by-product-sku' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Read the usage instructions in the description.
